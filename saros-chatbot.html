📋 Instrucciones para instalar en WordPress
  1. Obtené tu API Key gratis en aistudio.google.com (sin tarjeta)
  2. En WordPress: Apariencia → Editor de temas → footer.php
  3. Pegá el código del chatbot antes del </body>
  4. Reemplazá TU_API_KEY_AQUI con tu clave de Gemini
